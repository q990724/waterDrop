    根据机构名称搜索订单

    接口URL

    {{url}}/admin/order/search

    请求方式

    POST

    请求 Content-Type

    application/x-www-form-urlencoded

    请求Body参数

    参数 示例值 必填 参数描述
    val 水滴公益 必填 机构名称

    响应参数

    参数 参数描述
    code 200:成功 300:核心参数缺少 400:服务器错误或根据参数未找到指定数据
    msg 错误码说明
    oid 订单唯一id
    purpose 视频用途
    type 视频类型
    duration 视频时长
    cycle_start 开始制作时间
    cycle_end 结束制作时间
    company_id 订单机构唯一id
    status 订单状态码(1: 等待导演接受,2: 等待制作,3: 正在制作中,4: 制作完成,5: 取消制作)

    oc_number 订单机构编号
    oc_name 订单机构名称
    oc_type 订单机构类型
    oc_liaison 订单机构联系人
    oc_phone 订单机构联系电话
    oc_email 订单机构联系邮箱
